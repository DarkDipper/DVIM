# 💤 LazyVim

This nvim using [LazyVim](https://github.com/LazyVim/LazyVim) as starter pack.
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

    - Json

    - Markdown
    - Tailwind
    - Docker
